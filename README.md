# 🎓 Sistema de Gestão Acadêmica

Sistema desenvolvido para gerenciar alunos e turmas, aplicando conceitos de orientação a objetos e regras de negócio.

---

## 💻 Tecnologias
- Java  
- POO  
- Stream API  

---

## 🚀 Funcionalidades
- Cadastro de alunos e turmas  
- Listagem e edição de dados  
- Desativação de registros (soft delete)  
- Filtro de alunos por turma  

---

## 🗂️ Estrutura
- Aluno.java  
- Turma.java  
- AlunoService.java  
- TurmaService.java  
- Main.java  

---

## ⚙️ Conceitos aplicados
- Encapsulamento  
- Separação de responsabilidades  
- Validação de dados  
- Regras de negócio  

---

## ▶️ Como executar
1. Clone o repositório  
2. Abra na IDE  
3. Execute `Main.java`
